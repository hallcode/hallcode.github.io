The PTU was an organisation I volunteered for which helped represent and give advice to renters. I created a system of native custom elements (web components) to be used on their website and micro-sites such as their membership application form.

Tech stack

RPGUnit Test Dashboard

At my previous role at BGL I developed a dashboard to display the results of automated unit test runs.

The dashboard was part of the updated release process adopted by the department and changes could only be promoted up the environments if the test dashboard was green for all tests.

Sadly I don't have any screenshots of this one.

How it worked

The tests were run by a task scheduler on the company's IBM iSeries mainframe server, the tests and the software was written in the RPG language.

The results of the tests were output into a number of SQL database tables.

I wrote a PHP app which ran on the IBMi and ran the SQLs to get the test results for each test suite and the code coverage level.

Tech Stack

The dashboard used:

  • PHP for the backend
  • JavaScript (VueJS) for the frontend
  • SQL for the queries

Parrotts Fish and Chips

I created and published the website for Parrotts Fish and Chips.

They wanted a clean and simple design, which matched their existing print marketing materials.

The proprietors needed a website that was simple to update, but it also needed to be quite fast for me to design and code.

The solution was to create a basic site using PHP and Symfony components for routing etc, and then some YAML files which contain the menu information, contact details, and opening times, this way they can update the information easily by updating those files.

Github repo

Tech stack

For this project I used:

  • PHP (custom app built with Symfony components)
  • Front-end is static HTML and CSS, the menu uses minimal JS.
